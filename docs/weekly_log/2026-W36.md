## Key Results

### HIAD Sanity Check

| Window | Mean overlap with `U_drift` |
|---:|---:|
| `25` | `0.145` |
| `50` | `0.178` |
| `100` | `0.238` |
| `200` / full | `0.208` |

All tested windows stayed below the `0.3` pivot threshold. The default HIAD
window, `25`, had mean overlap `0.145`, confirming that raw RSSM update PCA is
not a good proxy for the true drift basis.

### HIAD First Run

| Metric | Change |
|---|---:|
| `J_slow` / `dc_trend` | `-14.2%` |
| `J_total` | `+22.6%` |
| `very_low` | `+61.1%` |
| `low` | `+68.1%` |
| `mid` | `+39.3%` |
| `high` | `+8.5%` |

The slow-band improvement was below the `15%` marginal threshold and came with
substantial error redistribution. Original HIAD is not viable.

### Post-Hoc Recurrent Damping

| eta | Mode | J_slow | J_total | Raw MSE |
|---:|---|---:|---:|---:|
| `0.20` | recurrent | `-19.8%` | `-15.2%` | `-19.8%` |
| `0.30` | recurrent | `-26.3%` | `-20.7%` | `-27.7%` |
| `0.50` | recurrent | `-32.4%` | `-27.5%` | `-39.6%` |

This was the Week 1 breakthrough. Inference-time recurrent damping works when
the basis is the diagnostic `U_drift` basis. Non-recurrent damping had almost
no effect on `J_slow`.

## Decisions

- Pivot away from self-estimating HIAD based on `Delta z_t` PCA.
- Keep inference-time recurrent damping as the viable intervention mechanism.
- Use diagnostic-derived or Adaptive-SMAD-derived bases for the next method
  iteration.
- Treat original HIAD as a useful negative result for the paper's
  intervention-limits narrative.
- Continue the controlled RunPod baseline plus anchor-only Adaptive-SMAD run
  to produce a fresh `current_U` basis.

## Current Interpretation

The failure was not damping itself. The failure was the basis. Raw transition
updates identify where the RSSM moves, while true drift identifies where the
open-loop trajectory separates from posterior latents. Those subspaces overlap
weakly.

The successful post-hoc recurrent run shows that a high-quality drift basis can
reduce slow and total latent error together. This shifts Month 8 from "can HIAD
self-estimate its basis?" to "how should we obtain the best basis for
inference-time recurrent damping?"
