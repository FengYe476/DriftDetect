## Key Decisions
- Mockable rollout protocol chosen over live agent coupling (enables unit testing, future V4 support)
- Mutable patch API added to img_step_patch (enables in-place U updates from scheduler)
- No EMA smoothing (B1 overlap 0.005-0.022 means successive bases are near-orthogonal, EMA adds complexity without benefit)
- Framing B recommended for Mismatch Finding (honest, reviewer-robust, connects to capacity-complexity mechanism)
